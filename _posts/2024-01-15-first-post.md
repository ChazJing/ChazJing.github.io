# 欢迎来到我的博客

这是我的第一篇文章！🎉

## 为什么开始写博客？

我一直想有一个地方可以记录自己的学习过程和思考。选择用 GitHub Pages + Jekyll 来搭建这个网站，因为它：

1. **免费** - 使用 GitHub 的免费托管服务
2. **简单** - 不需要复杂的数据库
3. **强大** - 支持 Markdown 写作

## 如何写文章？

在 `_posts` 文件夹中创建新的 Markdown 文件即可。文章顶部需要包含一些元信息：

```yaml
---
layout: post
title: 文章标题
date: 2024-01-15
categories: [分类1, 分类2]
tags: [标签1, 标签2]
---
```

## 文章正文

在这里写下你的内容。支持 Markdown 语法：

- **粗体** 和 *斜体*
- 列表和链接
- 代码块和图片
- 等等...
